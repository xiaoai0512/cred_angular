下拉框新旧版本修改点
1、字典下拉框
    旧版：div中：<select jf-select="StCd" ></select>
    新版：div中：<select jf-select="builder.option('StCd')"></select>

2、文本框和下拉框的上级class是要加wd190,不然必填校验会自动换行
如旧版：<div class="col3 float_left">
				<input class="layui-input" type="text" ng-model="userAndRoleRel.roleName" selector="selectRole" placeholder="请选择">
			</div>
    新版：<div class="wd190 float_left">
			<input class="layui-input" type="text" ng-model="userAndRoleRel.roleName" selector="selectRole" placeholder="请选择">
		</div>
