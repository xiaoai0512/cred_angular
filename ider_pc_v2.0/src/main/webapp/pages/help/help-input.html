文本框组件使用 - 入门指南
带格式文本框，金额类，银行卡类，日期控件，百分比数值。
1.在input框中，jf-format-input对应的指令为jfFormatInput，该指令封装了两种类型的实现。当要实现金额文本时传类型值为money，要实现银行卡文本时传类型值为bankCard。
2.日期控件主要实现的指令为jfDateInput，当要实现最小值和最大值的日期限制时，要往里面传条件。如实现最小值日期限制，jf-date-input="minDate"，在minDate对象上设置 属性min对应的日期值，如：$scope.minDate = {min : new Date()}; 要实现最大的日期值，则在对象上设置属性max对应的日期值，如：$scope.minDate = {max : new Date()};

js代码
    //日期设置
    $scope.minDate = {
        min : new Date(),//默认不限制，可缺省
        format :  'yyyy-MM-dd',//默认格式化年月日，可缺省
	    type : 'datetime' //默认只能选年月日，datetime为年月日时分秒,可缺省
    };
