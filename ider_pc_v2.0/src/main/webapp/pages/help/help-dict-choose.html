2.2.1 分组数据字典复选框
根据参数去后台请求数据字典，以复选框的方式进行展示。展示分两种：分组与未分组。
1.builder.option('AbvQtPcsgModCd'),去字典表里获取数据，参数'AbvQtPcsgModCd'为获取字典表里某一标识对应的字典键值。详细去看控制台dict请求的返回json。
2.builder.option('AbvQtPcsgModCd','1') 第二个参数对应数据字典中组数的第几组 。
html代码
     

          
          

                <input jf-checkbox="builder.option('AbvQtPcsgModCd')" name="checkbox2" ng-model="demo.checkbox2">
          

     

     
     

          
          

                <input jf-checkbox="builder.option('AbvQtPcsgModCd','1')" name="checkbox" ng-model="demo.checkbox">

2.2.2 动态组装数据字典复选框
1.按照规范的字段名[id, name]，对数据源进行自定义 2.自定义数据字典的数据源包括字段名，看如下代码与示例
html代码
     
     

            
            

                <input jf-checkbox="builder.option(demoArray)" name="checkbox3" ng-model="demo.checkbox3">
            

      

     
     

            
            

                <input jf-checkbox="builder.option(demoArray2,'cstId','cstNm')" name="checkbox4" ng-model="demo.checkbox4">
js代码
     //动态组装，自定义数据字典数据源
     $scope.demoArray = [{name : '字典1' ,id : '1'},{name : '字典2' ,id : '2'}] ;
     //按自定义字段组装数据字典
     $scope.demoArray2 = [{cstNm : '大客户',cstId : '1'},{cstNm : '小客户',cstId : '2'}] ;
