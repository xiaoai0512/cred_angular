列表显示字段扩展使用 - 入门指南
使用包含两部分
    1.字段过多其他字段默认隐藏，点击后再显示的操作.
    将隐藏字段写在class="title_messages"之后，详细写法看html代码
    框架保持7列字段不变形，根据需求使用这类说明
    2.显示使用数据字典转换显示出真实值
    ng-bind="字段名| jfConvert : 'dict' : '数据字典名称'"
     如：ng-bind="item.processDefineName | jfConvert : 'dict' : 'AbvQtPcsgModCd'"
