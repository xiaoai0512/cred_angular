表格组件使用 - 入门指南
本篇为你介绍表格的HTML使用，你将通过内置的自定义属性对其进行风格的多样化设定。请注意：这仅仅局限于呈现基础表格，如果你急切需要的是数据表格（datatable），那么你应该详细阅读：table模块.除了默认的表格风格外，我们还提供了其它几种风格，你可以按照实际需求自由设定
