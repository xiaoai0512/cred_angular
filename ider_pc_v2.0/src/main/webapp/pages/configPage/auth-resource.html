权限配置
在菜单管理中配置页面(按钮)资源，然后可以在角色管理分配选中角色的页面(按钮)的资源权限。
页面资源地址等同于路由配置文件中的路由地址
通过下面配置将demo-table.html页面的路由地址中被设置为：/tableDemo。则新增菜单管理时页面配置资源地址为：/tableDemo。
示例： 
  define(function(require, exports, module) {
     var config = [
        //客户信息管理 start	
         ['/tableDemo', 'demo/demo-table.html', 'demoController.js'],
          ];
      module.exports = config;
   });
按钮资源地址由"jf-permission"指令指定
在jf-permission="资源地址"指定按钮的资源地址
例如：在按钮的属性里添加jf-permission="demo.add",并将按钮资源地址设置为demo.add。
